# Platform-specific reference

Details for behavior and API usage unique to the  Platform Toolkit for Nintendo Switch™ package.

> [!Note]
> Refer to [Get started with Platform Toolkit for Nintendo Switch™ or Nintendo Switch™ 2](get-started-with-pt-and-switch.md) for detailed setup instructions.

## NMETA: Console account selection at startup

You might want to specify whether a user account is required when your application starts. Use the **Console account selection at startup** setting in the NMETA file to set this configuration. Depending on your selection, this might impact usage of the Platform Toolkit account APIs.
For more information, refer to [Console account selection at startup](https://developer.nintendo.com/html/online-docs/g1kr9vj6-en/Packages/SDK/NintendoSDK/Documents/Package/contents/Pages/Page_166503043.html#Anchor_166503043_h3_14).

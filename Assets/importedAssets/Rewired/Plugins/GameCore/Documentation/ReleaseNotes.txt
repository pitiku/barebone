Rewired Game Core Plugin

v36:
  - Bug Fix: Reverted breaking change made in v34 that inadvertently changed the minimum Rewired version to 1.1.61.0.

v35:
  - Added GDK 10.0.26100.6194 native library.

v34:
  - Removed all internal use of IGameInputReading::GetMotionState due to reported crashes by Microsoft with some hardware configurations.

v33:
  - Adapted to breaking changes in newest versions of Unity Game Core editor plugins preventing Rewired from detecting GDK version used by Unity.

v32:
  - Added GDK 10.0.26100.4020 native library.

v31:
  - Added hat switch to Flight Stick device profile.
  - Exposed product name for Racing Wheel devices. (Requires Rewired 1.1.59.0+)
  - Changed input polling to process entire remaining input history buffer on each update to address input latency issues with high refresh rate controllers (Power A Fusion Pro).

v30:
  - Added GDK 10.0.25398.4271 native library.
  - Changed how device profiles are handled so all elements can be read on devices that support multiple device profiles such as Racing Wheel.
  - Joystick Element Identifier tool now reports the actual number of buttons, axes, and hats reported by the GDK API and reports additional information.
  - Changed ControllerProfileType.Raw to ControllerProfileType.Generic.

v29:
  - Changed input polling to read next event from input history buffer instead of only the current event.

v28:
  - Reworked GDK version detection due to breaking changes in recent Unity Xbox Game Core versions.

  - Fixed manual GDK version selector not replacing native libraries when changing versions.
  - GDK version is now determined from the Unity extension rather than checking the GDK environment variable to detect which version of the native library to extract. If the system had a newer version of the GDK installed than the GDK version of the Unity editor being used, a library built to a newer version of the GDK could be installed causing errors in a build.

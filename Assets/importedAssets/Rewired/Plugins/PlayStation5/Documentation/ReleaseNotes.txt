Rewired PlayStation 5 Plugin

v7:
  - Added new input source option: Unity Input System PS5.
  - Changed minimum Rewired version to 1.1.63.0 for Unity 2021+.
  - Changed default gamepad vibration mode to Compatible2.

v6:
  - Bug Fix: Fixed inverted Y axis in orientation returned by PS5ControllerExtension.GetOrientation.

v5:
  - Added vibration renewal workaround to allow inifinite vibration.

v4:
  - Changed required minimum Rewired version to 1.1.47.0.
  - Added ControllerExtensions.IDualShockGamepadExtension interface to PS5GamepadExtension.
  - Reduced memory allocation overhead when setting multi-position trigger effects.

  - Added gamepad trigger effect functionality.
